# Setup

## Prerequisites
- Node.js + npm (or yarn; `yarn.lock` exists).

## Install
- `npm install`

## Environment variables and secrets
- `VITE_API_KEY` — RAWG API key (see `.env.example:1`).
- Do not commit real API keys; prefer `.env.local` for local dev.

## Run (development)
- `npm run dev`

## Build / preview
- `npm run build`
- `npm run preview`

## Lint
- `npm run lint`
